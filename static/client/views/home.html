About Solomon's House

Welcome to Solomon's House, the Civil Search Engine for Public Domain Educational Ebooks. This is not a piracy website, but rather a Search Wrapper like and for Google. You can contribute by seeding this magnetURI (53GB) with WebTorrent.

The Code

The database is Neo4J, which allows for Graph Search, and a Quantum-Graph Recommendation Engine called Mermaid. A Hieroglpyh is rendered as the recommendation button to seed the QRNG. We use VR-force-graph to display Graph Search to the client, including VR capability. The project is meant to be "Gazelle with Webtorrent," with an original educational use case. I have also developed a lightweight in-house javascript router called ANCHOR.js as an alternative to heavier frameworks like AngularJS and React.js. The code for website (a BitTorrent Indexer) is Open Source and can be found on Github. It is a Single-Page App (SPA).
